Frequently Asked Questions

Is OFJDBC really free?

Yes! OFJDBC is completely free and open-source. There are no licensing fees, subscriptions, or hidden costs.

What SQL query tools work with OFJDBC?

OFJDBC works with any JDBC-compatible database tool. DBeaver is the recommended choice and provides the best experienceâ€”it's free, full-featured, and works perfectly with OFJDBC. Other JDBC clients like IntelliJ IDEA Database Tools, Eclipse Data Tools, and JetBrains DataGrip also work.

How does OFJDBC compare to commercial SQL query tools for Oracle Fusion?

OFJDBC provides similar functionality to commercial tools but is free and open-source. It's perfect for consultants, small teams, or anyone who prefers using standard database tools like DBeaver.

Can I use OFJDBC instead of OTBI?

OFJDBC is ideal for ad-hoc SQL queries and data exploration. While OTBI has its place for creating formal reports, OFJDBC excels at quick troubleshooting and analysis tasks.

What are the AI-powered companion tools?

OFJDBC has two AI companions that help you work smarter: (1) Claude MCP Server - integrates with Claude AI for metadata-aware SQL development in your IDE, and (2) ChatGPT Assistant - a custom GPT for exploring Oracle Fusion metadata, business processes, and getting query templates. Both are free and complement OFJDBC perfectly.

Is OFJDBC secure?

Yes. OFJDBC connects directly from your machine to Oracle Fusion Cloud using your credentials. No data passes through third-party servers, and the code is fully transparent on GitHub.
